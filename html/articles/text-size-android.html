Adjust the Text Size on Android

Learn how to adjust the text size on Android.


Set the Device Zoom and Text Size

It may vary as to where to find the text size settings on your device depending on what device and software version you have. An easy way to find these settings is to search for Font Size or Text Size in the Settings search bar at the top or top right corner of the screen.

Google Pixel

  1. Open Settings
  2. Tap Display & Touch
  3. Tap Display Size and Text
  4. Adjust Font Size, Display Size, Bold Text and Outline Text Settings

Samsung Galaxy

  1. Open Settings
  2. Tap Display
  3. Do one of the following
    ‣ Tap Font Size and Style > Change the system font, enable Bold Text, or adjust the font size with the slider.
    ‣ Tap Screen Zoom > Adjust the display zoom with the slider.

Setup the Quick Settings Shortcut to Set Text Size

If you frequently change the text size on your device, you can add a button to the Quick Settings panel to easily adjust the text size.

  1. Open the Quick Settings panel by swiping down with one finger twice from the top of the screen or swipe down with two fingers once.
  2. Tap the pencil icon (found in the bottom right of the panel on Google Pixel or the top right on a Samsung Galaxy) to edit the controls.
  3. Find the Font Size button, then drag it into the Quick Settings panel.
  4. Tap the back button in the top right corner or look for a done or check button in the top right corner to save your changes.
  5. Tap Font Size, then tap the + or - to increase or decrease the text size.
  6. Tap Done to finish.
